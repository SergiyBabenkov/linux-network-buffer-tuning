Key Components:

  • VFS (Virtual File System): Provides a common interface for all file operations, including sockets
  • Socket Layer: Manages network endpoints with send/receive buffers for data transmission
  • Protocol Layers: TCP (reliable), UDP (fast), ICMP (control messages)
  • IP Layer: Handles packet routing and addressing across networks
  • Queueing Discipline (qdisc): Traffic control for scheduling and prioritization
  • Device Drivers: Hardware-specific code to communicate with NICs
  • NIC: Physical or virtual network interface hardware

Data Flow: Application → Libraries → System Calls → Kernel (VFS → Socket → Protocol → IP → qdisc → Driver) → NIC → Network
